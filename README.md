# kmd

Output the Kubernetes information to Markdown.

```sh
$ kmd get node
| NAME                       | STATUS | ROLES  | AGE | VERSION            |
| --                         | --     | --     | --  | --                 |
| ip-10-0-4-245.ec2.internal | Ready  | <none> | 18d | v1.16.8-eks-e16311 |

```
